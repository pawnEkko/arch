# 软件配置

* i3wm

```
~/.config/i3/config																				#i3的配置文件路径
```

* 必装字体

```
sudo pacman -S otf-font-awesome
yay -S ttf-font-awesome-5
yay -S ttf-meslo-nerd-font-powerlevel10k
```

* zsh
* picom混成器

```
sudo pacman -S picom              																#下载picom
~/.config/picom.conf		  																#picom的配置文件路径
https://github.com/pawnEkko/arch/blob/master/config/picom/picom.conf           					#我在GitHub保存的picom配置文件路径 
```

* polybar

```
sudo pacman -S polybar																			#下载polybar
~/.config/polybar                                 												#polybar的配置文件夹路径
#  配置文件夹中应该包含  config.ini - 配置文件   以及 launch.sh 启动文件
chmod +x ~/.config/polybar/launch.sh														#终端执行此命令以确保配置文件可以被i3执行
https://github.com/pawnEkko/arch/blob/master/config/polybar/								#GitHub保存的个人启动文件和配置文件路径
```
